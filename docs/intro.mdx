# Manuais

Um Manual de Procedimentos é como o GPS dos profissionais, só que em vez de te dizer para virar à direita ou à esquerda, ele te mostra os passos certinhos para fazer as coisas do jeito mais ninja.

## Vantagens dos manuais

Imagine que você está jogando um **novo jogo** de vídeo game. No começo, tudo parece confuso e você não sabe exatamente o que fazer. É como se você estivesse tentando montar um 
quebra-cabeça sem as instruções. Agora, pense em como seria mais fácil se você tivesse um manual que explicasse passo a passo **como jogar** o jogo, **quais botões apertar** e **como 
enfrentar os desafios**.

Da mesma forma, principalmente na Saúde, as coisas que precisam ser feitas de maneira **certa** e **organizada**. Essas coisas podem ser desde a maneira como um novo funcionário é 
treinado em um escritório até como consertar um aparelho quebrado em casa.

Aqui estão algumas vantagens de usar manuais de procedimentos:

1. **Clareza**. Os manuais fornecem instruções claras e detalhadas sobre como fazer as coisas. Eles ajudam a entender exatamente o que precisa ser feito, como um mapa que te guia.
2. **Consistência**. Imagine se todos estivessem jogando o mesmo jogo de vídeo game, mas cada pessoa usasse regras diferentes. Isso seria bem confuso! Manuais ajudam a garantir 
que as coisas sejam feitas da **mesma** maneira por **todos**, o que é importante para um bom resultado.
3. **Eficiência**. Ter um manual significa que você **não precisa perder tempo tentando descobrir** como fazer algo toda vez. Você pode simplesmente seguir as instruções e fazer as 
coisas de forma mais rápida e eficiente.
4. **Treinamento mais Fácil**. Para alguém novo em alguma atividade, como um **novo** **funcionário** em uma empresa, os manuais são como guias que ensinam passo a passo. Isso torna 
o treinamento mais fácil e rápido.
5. **Lembrança**. Às vezes, esquecemos como fazer coisas que não fazemos com frequência. Um manual serve como um lembrete útil para nos ajudar a lembrar dos passos corretos.
6. **Padrões de Qualidade**. Manuais ajudam a manter um alto padrão de qualidade. Quando todos seguem as mesmas instruções, é mais provável que o resultado seja bom.
7. **Resolução de Problemas**. Se algo der errado, um manual pode conter **soluções** para problemas comuns. Isso pode **poupar** **tempo** e frustração ao tentar encontrar uma solução.
8. **Compartilhamento de Conhecimento**. Imagine se você descobrisse uma maneira mais fácil de jogar um jogo e pudesse compartilhá-la com seus amigos. Manuais permitem que o 
conhecimento seja compartilhado com outras pessoas.

Então, assim como um manual de jogo torna a jogabilidade mais divertida e organizada, manuais de procedimentos tornam várias atividades na vida mais fáceis, organizadas e eficientes!

### What you'll need

- [Node.js](https://nodejs.org/en/download/) version 16.14 or above:
  - When installing Node.js, you are recommended to check all checkboxes related to dependencies.

## Generate a new site

Generate a new Docusaurus site using the **classic template**.

The classic template will automatically be added to your project after you run the command:

```bash
npm init docusaurus@latest my-website classic
```

You can type this command into Command Prompt, Powershell, Terminal, or any other integrated terminal of your code editor.

The command also installs all necessary dependencies you need to run Docusaurus.

## Start your site

Run the development server:

```bash
cd my-website
npm run start
```

The `cd` command changes the directory you're working with. In order to work with your newly created Docusaurus site, you'll need to navigate the terminal there.

The `npm run start` command builds your website locally and serves it through a development server, ready for you to view at http://localhost:3000/.

Open `docs/intro.md` (this page) and edit some lines: the site **reloads automatically** and displays your changes.
